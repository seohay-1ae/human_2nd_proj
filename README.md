# DB.jar

## 프로젝트 개요

2차 프로젝트 주제: 공공데이터를 활용한 여행지 추천 및 여행 기록 공유 커뮤니티 서비스

**DB.jar**는 공공데이터 api를 활용한 여행지 정보 제공 서비스로,  
특히 뱃지와 아바타 시스템을 접목하여 사용자 흥미를 유발하는 방법에 초점을 맞추어 개발한 프로젝트입니다.
